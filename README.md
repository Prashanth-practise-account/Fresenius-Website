# Project Overview
Fresinus.com is a corporate-style website inspired by Fresenius, designed to showcase healthcare divisions, products, and stock information. It includes a homepage slider, product sections, stock info bar, and smooth section navigation with up/down arrows.


# Tech Stack
1.Frontend: React.js, HTML5, CSS3, JavaScript (ES6+)

2.Styling: Custom CSS 

3.Deployment: GitHub 

# Features

1.Homepage slider with image descriptions

2.Section-based navigation with up/down buttons

3.External links to Fresenius resources (share price, product divisions)

4.Responsive design (works on desktop & mobile)

5.Smooth scrolling between sections

# To Run The Project

1. # Go into project folder
cd fresinus.com

2. # Install dependencies
npm install

3. # Run development server
npm start
